Retos en ANSI C - Sistemas Operativos

Cinco ejercicios en C para practicar conceptos de Sistemas Operativos.
Fuentes: challenge1.c … challenge5.c.
Ejecutables: reto1 … reto5.

Requisitos
gcc
make
Linux o macOS

Compilación

Compilar todo:
make

Ejecución
Después de compilar:
./reto1
./reto2
./reto3
./reto4
./reto5

Limpieza
make clean
